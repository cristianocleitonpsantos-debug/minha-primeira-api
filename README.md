# 📚 Minha Primeira API com Flask

Este projeto é uma API simples desenvolvida com **Flask** para gerenciamento básico de alunos. Ela permite listar todos os alunos e buscar um aluno específico pelo ID.

---

## 🚀 Tecnologias Utilizadas

- Python 3
- Flask

---

---

## 📦 Endpoints disponíveis:

- GET /
- GET /health
- GET /alunos
- GET /alunos/<*id*>
